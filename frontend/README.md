# Frontend

Mock-first cinematic workbench prototype for `po1market`.

## Current direction

- workbench-first first screen
- asymmetric bento layout
- editorial sans + monospace + light pixel accents
- mock data only (no live backend in this milestone)

## Run

```bash
cd frontend
npm install
npm run dev
```

## Test

```bash
cd frontend
npm run test
```

## Scope

- uses local mock data
- demonstrates the product shell and single-page interaction flow
- targets future integration with `POST /api/v1/recommendations`
